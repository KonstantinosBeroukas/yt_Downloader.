# YouTube Media Downloader

A robust Python-based utility designed to download high-quality video and audio content from YouTube. This project focuses on integrating external libraries and managing file system operations.

## 🚀 Features
* **Format Selection:** Choose between downloading full video (MP4) or audio-only (MP3) formats.
* **Resolution Support:** Options for 720p, 1080p, and 4K (where available).
* **Automated File Naming:** Sanitizes video titles to ensure compatibility with Windows, macOS, and Linux file systems.
* **Progress Tracking:** Real-time feedback on download percentage and file size.

## 🛠️ Tech Stack
* **Language:** Python 3.x
* **Core Library:** `pytube` or `yt-dlp` (High-performance media handling).
* **Error Handling:** Robust validation for invalid URLs and connection timeouts.
